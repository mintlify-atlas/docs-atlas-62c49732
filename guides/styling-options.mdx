## Overview

Draw Folder Structure offers 20+ distinct visual styles to render your folder and file structures. Each style uses different characters and symbols to create unique visual representations.

## Available Styles

The extension provides styles ranging from classic tree views to emoji-enhanced designs. Choose the style that best fits your documentation aesthetic.

### Classic Styles

<AccordionGroup>
  <Accordion title="ClassicDashes" defaultOpen>
    Traditional tree structure with dashes and pipes.

    **Example Output:**
    ```
    project
      └── src
      ├── index.ts
      └── utils.ts
    ```

    **Best for:** Traditional documentation, technical docs, README files
  </Accordion>

  <Accordion title="PipesAndHyphens">
    Clean structure using pipes and hyphens.

    **Example Output:**
    ```
    project
      |- src
      |- index.ts
      |- utils.ts
    ```

    **Best for:** Technical documentation, minimalist style
  </Accordion>
</AccordionGroup>

## Changing Your Style

There are multiple ways to change the style setting:

<Tabs>
  <Tab title="Settings UI">
    1. Open VS Code Settings (`Ctrl+,` or `Cmd+,`)
    2. Search for "draw.folder.structure.style"
    3. Select your preferred style from the dropdown
  </Tab>
  <Tab title="settings.json">
    Add or modify the style setting in your `settings.json`:

    ```json
    {
      "draw.folder.structure.style": "ClassicDashes"
    }
    ```
  </Tab>
  <Tab title="Command Palette">
    1. Open Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`)
    2. Type "Preferences: Open Settings (UI)"
    3. Search for "draw.folder.structure"
    4. Change the Style option
  </Tab>
</Tabs>

## Style Examples in Context

Here's how different styles render the same project structure:

<CodeGroup>
```text ClassicDashes
my-project
  └── src
  ├── components
  ├── Button.tsx
  └── Card.tsx
  ├── utils
  └── helpers.ts
  └── index.ts
```

```text EmojiDashes
my-project
  └── 📁 src
  ├── 📁 components
  ├── Button.tsx
  └── Card.tsx
  ├── 📁 utils
  └── helpers.ts
  └── index.ts
```

```text MinimalistDots
my-project
  • src
  • components
  • Button.tsx
  • Card.tsx
  • utils
  • helpers.ts
  • index.ts
```

```text NestedCircles
my-project
  ◉ src
  ◉ components
  ○ Button.tsx
  ○ Card.tsx
  ◉ utils
  ○ helpers.ts
  ○ index.ts
```
</CodeGroup>

<Note>
  The extension automatically wraps the generated structure in Markdown code blocks (\`\`\`) for proper formatting.
</Note>
